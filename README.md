# Pass Bindings for Ruby

Pass bindings for ruby - largely based on [stripe-ruby](https://github.com/stripe/stripe-ruby).

## Installation

To build the gem from source, run:

```
gem build pass.gemspec
```

## Requirements

* Ruby 1.8.7 or above
* rest-client, multi_json
